History Multiplayer

Game Multiplayer FPS (first person shooter) pertama kali menjadi berita utama pada tahun 1996 ketika sebuah perusahaan bernama id Software, yang terkenal dengan game DOOM dan Commander Keen-nya, merilis apa yang dianggap sebagai fenomena game sebagai versi uji multipemain dari game Quake-nya.

Quake menggunakan mesin game 3-D yang memberikan kemampuan kepada pemain untuk memainkan game FPS di dunia 3-D yang sebenarnya di mana objek dalam game dapat dilihat oleh pemain dari sudut mana pun. Sama menakjubkannya adalah teknologi yang memungkinkan para gamer untuk memainkan game deathmatch multipemain melalui Internet. Bermain internet dimungkinkan melalui penggunaan arsitektur klien/server. Siapa pun dapat menjalankan server di komputer, yang akan melacak lingkungan game, fisika, dan pemain dalam game. Pemain lain dengan versi game yang sama dapat terhubung ke server Quake dan voila multiplayer deathmatch!

Secara alami, koneksi game memiliki masalah. Misalnya, seorang pemain di Kanada Timur dengan modem 56kps yang bermain bahkan di server T1 yang cepat di bagian Barat Amerika Serikat, misalnya, akan mengalami kelambatan yang mengerikan dalam permainan sehingga hampir tidak dapat dimainkan. Namun, dengan rilis resmi Quake, id Software telah mengatasi beberapa dari banyak masalah bandwidth dalam game, tetapi seberapa cepat dan lancar permainan multipemain Anda berjalan sangat bergantung pada koneksi Internet Anda dan lokasi server tempat Anda bergabung.

Selain kemajuan teknologi yang ditawarkan oleh Quake dan game-game berikutnya yang mengikutinya, wajah game sebagian besar berubah karena sifat interaktif dari game multipemain dan keuntungan komunikasi dari Internet itu sendiri. Tiba-tiba orang bisa pergi ke toko dan membeli game multipemain, pulang ke rumah dan online dengannya dan mendapati diri mereka terlibat dalam aspek sosial dan komunitas dari game tersebut. Game-game ini tidak hanya menawarkan Anda kesempatan untuk bermain head-to-head melawan orang lain, tetapi juga memungkinkan Anda untuk bergabung dengan tim dan bermain multiplayer kooperatif

Kelompok pemain akan bersatu sebagai sebuah tim, yang disebut Klan, dan menantang klan lain melalui Internet. Jumlah pecahan akan diposkan, orang-orang akan bercanda bolak-balik di ruang obrolan IRC dan di forum situs Web yang membawa aspek sosial yang sama sekali baru ke permainan multipemain. Citra seorang gamer juga berubah seiring berjalannya waktu. Pandangan stereotip seorang gamer adalah seorang remaja laki-laki dengan joystick di tangannya, tetapi seiring waktu orang mulai menerima gagasan bahwa orang-orang dari semua lapisan masyarakat; profesional, pelajar dan bahkan wanita memainkan permainan ini.

Sementara pertengahan hingga akhir 90-an tampaknya menjadi waktu di mana game memanfaatkan teknologi dan cara gamer bermain, ini tidak hanya terjadi pada game FPS populer. Game seperti game balap Interstate '78 Activision dan judul RPG Diablo Blizzard, misalnya, sedang populer. Judul-judul ini bersama dengan banyak lainnya menekankan pada kemampuan multipemain Internet. Untuk gamer di Internet, ini meningkatkan nilai gameplay game. Anda tidak hanya dapat memainkan game pemain tunggal dengan durasi penuh, tetapi Anda juga dapat memperpanjang umur game dengan bermain bersama teman dan keluarga di LAN atau terhubung ke server Internet untuk bersaing dengan yang lain.

Game Multiplayer Terbesar

Di kalangan game ada beberapa perdebatan tentang siapa yang memulai fenomena multipemain. Banyak gamer, sebagian besar yang memiliki hasrat untuk game FPS, menganggap Quake Software sebagai game yang memulai semuanya. Namun, di dunia game RPG, Anda akan menemukan bahwa kredit diberikan kepada Origin untuk membuka tes alfa Ultima Online pada tahun 1996.

MMOG (Massively Multiplayer Online Games) dan MMORPG (Massively Multiplayer Online Role-Playing Games) hadir dan para gamer dengan cepat beradaptasi dengan konsep pembelian sebuah game dan kemudian membayar bulanan untuk berlangganan agar dapat terus memainkan game tersebut. MMORPG berlangsung di dunia game yang gigih, di mana gameplay dan dunia game itu sendiri terus berkembang dan berubah. Untuk tujuan ini, pengembang game akan terus memperbarui server game dan gamer membayar harga dari dunia yang terus-menerus ini melalui biaya bulanan tambahan.

MMOG benar-benar masif dan memungkinkan ribuan gamer untuk bergabung dengan dunia game dan secara bersamaan berinteraksi dalam game itu. Saat Anda bergabung dengan permainan, Anda terus bermain, terlepas dari siapa lagi yang aktif pada saat yang sama dengan Anda. Tidak seperti kebanyakan game, MMORPG tidak menawarkan akhir. Tujuan Anda bukanlah bermain game untuk mengalahkannya. Sebaliknya itu dapat dibandingkan dengan dunia yang tidak pernah berakhir yang, jika permainan bertahan di pasar, Anda benar-benar bisa bermain selama bertahun-tahun dengan karakter yang sama. Karakter yang tumbuh, mempelajari keterampilan baru, dan memperoleh item baru selama Anda bermain. Setiap server game dalam MMORPG biasanya menampung setidaknya seribu pemain dan server menjalankan lingkungan untuk bagian tertentu dari dunia game. MMORPG yang baik akan dibuat dari beberapa server, masing-masing menyediakan bagian dari dunia dan akan memungkinkan gamer untuk melintasi antara server pada dasarnya, memungkinkan gamer untuk melakukan perjalanan ke berbagai belahan dunia game.

Sama seperti pembentukan klan untuk game penembak multipemain yang populer, aspek komunitas MMORPG disatukan melalui gamer yang membuat guild dan membuat aliansi dan teman di dalam game. Inti dari MMORPG didasarkan pada aspek sosial dan permainan sering kali menawarkan dukungan dan bantuan tambahan untuk klan dan serikat yang terbentuk dalam permainan ini. Akar MMORPG dan MMOG diyakini berasal dari MUD, Multi-User Dungeon, dan game multipemain berbasis browser.

MULTIPLAYER INTERNET GAMES

Bagi para gamer yang tidak tertarik untuk menembak, mengambil kendali taktis pasukan, atau menyelesaikan misi di dunia fantasi, masih ada cara lain untuk terlibat dalam game multipemain. Game multipemain internet adalah game yang memerlukan koneksi ke Internet untuk bermain, dan merupakan jenis game PC. Game internet adalah game online saja yang Anda mainkan di browser Anda tanpa harus membeli perangkat lunak. Server game yang Anda sambungkan disediakan oleh layanan game seperti MSN Gaming Zone, Yahoo! Games, dan Pogo untuk menyebutkan beberapa situs Web game Internet. Untuk bermain di layanan ini, Anda biasanya harus membuat akun di situs Web dan masuk untuk bermain. Sebagian besar layanan ini menawarkan akun secara gratis (walaupun Anda mungkin harus melihat satu atau dua iklan), dan mungkin menawarkan fitur dan permainan eksklusif jika Anda memutuskan untuk membayar langganan layanan tersebut.

Game internet menawarkan kesempatan kepada gamer kasual untuk menjadi bagian dari komunitas game tanpa harus menghabiskan berjam-jam setiap hari untuk bermain pada satu waktu. Di berbagai situs game, Anda akan menemukan genre game yang mencakup arcade, kartu, kasino, olahraga, dan bahkan game strategi dan aksi mini. Layanan game Internet akan melacak statistik Anda dan menawarkan forum dan ruang obrolan dalam game untuk berkomunikasi dengan pemain lain dalam game Anda. Anda juga dapat membuat daftar teman dalam layanan game yang akan memberi tahu Anda ketika seorang teman masuk dan ruang permainan apa yang mereka mainkan sehingga Anda dapat dengan mudah bergabung dengan mereka.

GAME PC MULTIPLAYER JALAN MASA DEPAN

Banyak game PC di pasaran saat ini menawarkan beberapa jenis kemampuan multipemain di mana Anda dapat bermain di rumah di LAN atau melalui Internet dalam deathmatch atau kompetisi head-to-head. Itu tergantung pada gim itu sendiri tetapi beberapa bahkan akan memiliki opsi yang memungkinkan Anda bermain multipemain kooperatif melawan orang lain atau pemain AI komputer. Gim itu sendiri biasanya akan membawa Anda selangkah demi selangkah melalui proses mengonfigurasi gim Anda untuk multipemain dan menawarkan cara untuk mencari server gim langsung dari menu utama gim.

Banyak gamer lebih memilih game multipemain karena fakta sederhana bahwa game ini umumnya akan menawarkan game pemain tunggal ditambah kemampuan multipemain menambahkan gameplay baru dan nilai replay ke game. Bagi para gamer kasual di luar sana yang tidak tertarik menghabiskan uang dan waktu untuk game multipemain yang berdiri sendiri, popularitas layanan game Internet menawarkan banyak pilihan dan suasana sosial untuk bermain game.

Terlepas dari jenis game yang Anda suka mainkan di dunia game yang terus berubah saat ini, tidak ada keraguan bahwa Anda dapat menemukan game multipemain yang cocok untuk Anda!
